# 🎬 Netflix Content Catalog Analysis

An Exploratory Data Analysis (EDA) dashboard examining global streaming trends, content format distributions, production hubs, and specific cast/director queries across **8,800+ Netflix titles**.

## 📌 Project Overview & Key Queries

This analysis explores the Netflix titles dataset using Python Pandas to uncover strategic streaming patterns and catalog dynamics.

### 🔍 Key Insights Covered:
- **Format Distribution:** Comparative ratio of Movies vs. TV Shows in the library.
- **Geographic Production Hubs:** Top 10 content-producing nations (US, India, UK, etc.).
- **Director Portfolios:** Analysis of prolific creators like *Rajiv Chilaka* and *Marcus Raboy*.
- **Filmography Queries:** Search & extraction of titles starring *Shah Rukh Khan* and *David Attenborough*.
- **Duration Trends:** Average runtime metrics and category distribution.

---

## 🛠️ Tech Stack & Tools Used
